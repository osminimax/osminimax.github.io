현재 크롬 웹 스토어에서 구버전인 Manifest V2를 사용하는 익스텐션들은 전부 설치가 불가능하게 막혀 있다.

신규 설치 차단을 뚫고 익스텐션을 강제 설치하는 법을 알아보자.

현재 chrome 웹 스토어에서 Manifest V2를 사용하는 모든 익스텐션들은 이렇게 신규 설치가 아예 안되도록 막혀 있다.

그러나 옛날에 개발된 대부분의 익스텐션들은, 특히 지속적인 유지보수가 안 되는 익스텐션들은 버전업이 안 되어 막혀버린 경우가 많다.

심지어 같은 기능을 하는 새로운 익스텐션이 없다면 그저 ‘옛날엔 이런 기능이 있었는데…’ 하며 손가락을 빨 수밖에 없는 상황이 되어버린 것이다.

이렇게 익스텐션 페이지에서 chrome에 추가 버튼이 비활성화 되어 있고, 클릭이 불가능하다.

아래 안내의 자세히 알아보기를 클릭해도 아무런 영양가가 없다.

그런데 오늘 혹시나 해서 해봤더니 신규 설치에 성공했다. 그냥 너무 쉽게 뚫었다.

현재까지 애용하고 있는 Copy All Urls 익스텐션을 기준으로 설명한다. 유블록 오리진 같은 것도 동일한 방법으로 신규 설치 가능하다.

# 방법

  1. F12 혹은 Ctrl + Shift + I를 눌러 개발자 도구를 열고 Elements 탭에 들어간다.

  2. Element 선택 도구(단축키 Ctrl + Shift + C)를 눌러 활성화한다.

  1. 회색으로 비활성화된 ’Chrome에 추가’버튼을 누른다. div를 잡으면 왼쪽의 작은 삼각형을 눌러 하위 태그를 보면 button 태그를 찾을 수 있다.

  2. 라인에서 disabled 라는 단어를 더블클릭해 지워준다.

  1. 파랗게 활성화된 버튼을 클릭해 익스텐션을 설치한다.

백엔드에서 차단 안 해준 구글 직원분께 큰 감사를 보냅니다.
