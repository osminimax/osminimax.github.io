# 작성 목표(+25.02.18)

좋은 블로그 내용들에 따르면,

  1. 직접 측정하고 검증한 자료
  2. 상세한 개발 기록
  3. 읽기 쉽게 정리한 자료
  4. 기술적 트러블슈트

이라고 하는데, 결론은 새로운 정보, 정보의 정제, 재현가능성 이라는 근본적인 방법에 맞추어 열심히 적어보도록 하겠습니다.
